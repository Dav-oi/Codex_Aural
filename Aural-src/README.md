# Aural — 让 Codex 开口说话！！！
[Dave-oioioi/Codex_Aural](https://github.com/Dave-oioioi/Codex_Aural)

---

## 【功能概述】

1. Aural 言简意赅 而非死板朗读！
2. Edge TTS（默认） — 微软 Edge 在线语音合成，音质好，需联网
3. Windows System TTS — 系统内置语音，离线可用，响应快

---

## 【使用说明】

安装后重启终端，Codex 每条回复自动朗读。

```powershell
tts "你好世界"     # 手动朗读
tts-last           # 重听上一条
TTS                # 对话中重听
```
