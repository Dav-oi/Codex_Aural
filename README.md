Codex 相关文本项目工作区。

## 项目

### Aural — Codex 全局语音朗读
Edge TTS 晓晓神经语音，让 Codex 所有对话自动朗读回复。

- 单文件安装包 `Aural-Setup-v1.0.exe`
- 后台即用即走，临时文件自动清理
- 支持 `tts` / `tts-last` 快捷命令

### Programs — 开发工具
常用开发工具的下载地址索引。
